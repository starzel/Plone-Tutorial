plonekonf.talk Installation
---------------------------

To install plonekonf.talk using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``plonekonf.talk`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        plonekonf.talk
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
